This project is a fullstack task management application built to practice and demonstrate real-world web development skills. 

The backend is developed using Express.js with MySQL as the database, providing RESTful APIs for authentication and task management. JWT is used to handle secure user authentication.

On the frontend side, the application is built with React and styled using Tailwind CSS to create a clean and responsive user interface.

Users can register, log in, and manage their tasks with features such as creating, updating, and deleting tasks. Each user can only access their own data, ensuring basic data security.

This project focuses on understanding the integration between frontend and backend, handling authentication, and building a structured and maintainable fullstack application.
